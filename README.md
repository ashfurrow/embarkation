# Embarkation

This is a fork of the [Embark colour theme](https://embark-theme.github.io) for VSCode. I'm publishing this mainly so I can easily use this colour scheme across multiple computers. You're welcome to use it too.

## Why not contribute to the project instead?

[They have a VSCode version](https://github.com/embark-theme/vscode) but it appears to be a stalled a work-in-progress. [The main contributor](https://github.com/embark-theme/vscode/pull/1#issuecomment-888684024) isn't prioritizing VSCode, but _is_ open to someone else contributing it. However, this is my first colour scheme and I'm not interested in the extra work that comes with OSS ownership at the moment.
